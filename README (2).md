# Kazi Mahir Adeeb — Personal Portfolio

**Live:** [adeeb13.github.io/Kazi-Mahir-Adeeb](https://adeeb13.github.io/Kazi-Mahir-Adeeb)

## Deploy to GitHub Pages

1. Push this entire folder to a repository named `Kazi-Mahir-Adeeb`
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Your site will be live in ~2 minutes

## Structure

```
/
├── index.html          ← Main portfolio website
├── css/
│   └── style.css       ← All styles
├── js/
│   └── main.js         ← Cosmos, particles, interactions
├── cv/
│   └── index.html      ← Beautiful HTML CV (print-ready)
└── README.md
```

## CV

The CV is at `/cv/index.html` — open in browser and **Ctrl+P → Save as PDF** for a perfect print-quality CV.
